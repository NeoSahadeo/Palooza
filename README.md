## Documentation

Palooza uses TypeDoc for its documentation. To read it the
docs clone this repository,

```bash
git clone --depth=1 https://github.com/NeoSahadeo/Palooza
cd Palooza
```

then run `pnpm read` and open `http://127.0.0.1:8080/`
